# CLinkedin

[![CI Status](https://github.com/se-edu/addressbook-level3/workflows/Java%20CI/badge.svg)](https://github.com/se-edu/addressbook-level3/actions)
[![codecov](https://codecov.io/gh/se-edu/addressbook-level3/branch/master/graph/badge.svg)](https://codecov.io/gh/se-edu/addressbook-level3)

![Ui](images/Ui.png)

**CLinkedin is a desktop application for managing your contact, optimized for use via a Command Line Interface (CLI).** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface). If you can type fast, CLinkedin can get your contact management tasks done faster than traditional GUI apps.

* If you are interested in using CLinkedin, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing CLinkedin, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
